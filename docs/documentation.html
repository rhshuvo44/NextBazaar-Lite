🚀 Features

  • Tailwind CSS (CDN)
  • Responsive design
  • Dark mode switcher
  • Swiper.js slider
  • Mobile menu toggle
  • SEO optimized meta tags
  • 404 page included

🔧 How to Use

  1. Download and unzip the template.
  2. Open index.html in a browser.
  3. Edit content in the HTML files as needed.
  4. Change styles in assets/css/styles.css.
  5. Customize behavior in assets/js/main.js.
